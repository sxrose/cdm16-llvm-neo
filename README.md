# Neo CdM-16 LLVM backend

Это репозиторий проекта "Neo CdM-16 LLVM backend", здесь мы занимаемся доработкой и поддержкой компиляторной инфраструктуры LLVM для CdM-16. Проект является продолжением [работы которая была начата задолго до нас](https://github.com/leadpogrommer/llvm-project-cdm).  

Мы стараемся поддерживать свежие версии LLVM, фиксить баги, добавлять фичи (языковые и для инструментов), включать новые оптимизации, а также улучшать качество кодовой базы.

## Компиляторы

Сейчас мы поддерживаем компилятор (freestanding подмножества) языка C (clang), и в активной разработке находится [Rust инструментарий для платформы](https://github.com/ylab-nsu/cdm16-rust).
